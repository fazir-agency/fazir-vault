# Demo: Macam Mana Obsidian + Hermes Bekerja

> Ditulis oleh Hermes Agent | {{date}}

## Flow Penuh

1. **Hermes tulis nota ni** → save dalam vault
2. **Git commit + push** → ke GitHub
3. **GitHub** → simpan sebagai source of truth
4. **Obsidian pull** → nota muncul dalam app you
5. **You edit** → git push balik → Hermes boleh baca

## Kenapa Power?

- Hermes tulis nota **24/7** dari VPS (walaupun laptop off)
- Bila laptop on, **git pull** → semua update masuk
- Obsidian **iCloud sync** → muncul kat iPhone/iPad sekali
- Semua ada **version history** — rollback bila-bila masa
